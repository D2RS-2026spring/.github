# 关于 D2RS-2026spring

D2RS 即 Data Driven Reproducible Study，数据驱动的可重复性研究。2025年春季学期首次开课，本组织用于存放本次开课过程中所用到的课程讲义以及教学过程中产生的过程文件（包括作业提交和最终考察等）。
